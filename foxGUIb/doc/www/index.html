foxGUIb
... is an interactive gui builder and codegenerator for FXRuby. It is a convenient tool that makes it easy to quickly build complex and good looking graphical user interfaces for Ruby. 

download: all releases of foxGUIb

users guide: thanks to my dear friend Mike Parr there is a comprehensive User's Guide available.

screen shot: the window in the back is the main window of foxGUIb showing the "widget tree" the "widget selector" and the "properties" of the currently highlighted widget (blue). The window in the front is the design window, where a user has built a simple user interface.

installation v0.6.2:

  1. install libGUIb: download the latest version of the package libGUIb-install and uncompress it to a temporary directory. go to that directory and run ruby install.rb (on linux this requires superuser privileges)
  2. install foxGUIb: download the latest foxGUIb package and uncompress it to a directory of your choice.  
  3. you are done. 

installation v0.7:

the release process was too complicated with 3 packages, so i simplified it:
dowload the all-in-one-archive and install libGUIb. see installation.txt for details.

usage:

  • go to the foxGUIb directory and run foxGUIb either by:
    • double-clicking on foxGUIb-win32.rbw (on windows)
    • typing ruby foxGUIb.rb (on *nix-like systems)
  • to release files generated by foxGUIb you might want to distribute them together with libGUIb14.rb, which is also available for download as single file (libGUIb14.rb-redistributable*.zip).

mailing list: foxGUIb users may use the fxruby-users mailing list (low traffic).
